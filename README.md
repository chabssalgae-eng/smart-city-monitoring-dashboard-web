# Smart City Monitoring Dashboard

## Project Overview
A Smart City facility monitoring dashboard created using QGIS and modern web technologies.

## Features
- Interactive OpenStreetMap
- Facility monitoring dashboard
- Real-time facility status
- Alert management
- Facility statistics
- System health visualization

## Technologies
- QGIS
- HTML
- CSS
- JavaScript
- Leaflet.js
- Chart.js

## Development
This project was developed using ChatGPT and Cursor AI to rapidly prototype an interactive GIS monitoring dashboard based on QGIS facility data.
